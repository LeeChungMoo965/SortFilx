## 유튜브 API발급 받는 방법
> 발급 받은뒤 따로 저장해놓으세요~
> 저장 하셨다면 프로젝트 최상위 폴더에 .env 파일을 만들고 VITE_YOUTUBE_API_KEY="여기에 발급 받은 유튜브 api키 입력" 입력하시고 저장하면 됩니다!!
[유튜브 API 발급 방법] (https://boleesystem.tistory.com/996)


## 프로젝트 실행방법(터미널에 입력하시면 됩니다~)
```bash
cd shortflix-app
```
```bash
npm run dev
```
